InterProScan
========

Contains useful scripts for genomics.

filter_InterProScan.pl	- Parses InterProScan for GOs and other relevant information
filter_GOs.pl           - Parses the output of filter_InterProScan.pl and generates two lists: One to be 
                          used as input for cateGOrizer, the other one a simple list of the proteins and
                          their GOs
